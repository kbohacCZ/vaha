Moje váha

Nejprve propoj aplikaci s GitHubem.
Data se budou ukládat do tvého soukromého Gistu.

Krok 1

Vytvoř přístupový token

Otevři v prohlížeči tuto stránku a přihlas se svým GitHub účtem:

🔗 github.com/settings/tokens/new

Vyplň Note (např. moje-vaha), nastav Expiration na No expiration a zaškrtni pouze oprávnění gist. Pak klikni na Generate token a zkopíruj vygenerovaný token (začíná ghp_).

Krok 2

Zadej token do aplikace

Vlož token sem. Aplikace ho uloží pouze v tomto prohlížeči a automaticky vytvoří soukromý Gist pro tvoje data.

Informace

Kde budou data uložena?

Aplikace vytvoří v tvém GitHub účtu soukromý Gist s názvem vaha_data.json. Soubor je viditelný pouze tobě. Token zůstane uložen v localStorage tohoto prohlížeče.
